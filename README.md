# UFCProject

To add the latest event you need to run:

    main.py 
    dataCleaningV2.py

By doing this, the following things will be updated in DataFiles2 folder: 

    CleanData.csv 
    FightTotals.csv
    MasterFightList.csv
    UFC_Fighters.csv

In the Events2 folder, the following things get updated:

    event names.txt
    MasterFightList_#.csv
